Short Summary

Privacy First CleanLite is designed to be privacy-friendly. The app does not collect, transmit, or store personal user data on our servers. All cleaning and analysis is performed locally on the device.

What Data the App Uses

CleanLite operates only with files and resources on the device, and with limited system APIs needed to free storage and memory. We do not collect or transmit any personal information, identifiers, analytics, or usage telemetry.

  • Local file access: Scans storage and cache folders to find junk files. Deletes only after user confirmation.
  • System stats: Reads RAM usage and free storage to show in the UI. Not sent anywhere.
  • Permissions: Requests only minimal required permissions for cleaning tasks.

Permissions We May Request

  • Read/Write External Storage — to locate and delete junk files.
  • Background Run — for Quick Clean widget (if enabled).

You can review/revoke permissions in device settings. The app will function within granted permissions.

No Tracking, No Analytics

Privacy Focused No analytics, trackers, or telemetry are included in the initial release.

Data Retention

No data is retained. Temporary processing files are deleted immediately after cleaning.

Third-party Services

No external services, APIs, or ads in version 1. If added in future, consent will be required and this policy updated.

Children

Not directed to children under 13. No data knowingly collected from minors.

Security

Operates locally. Users are responsible for securing their device (lock screen, encryption, etc.).

Changes to this Policy

We may update this policy as the app evolves. Any changes will be posted here.
